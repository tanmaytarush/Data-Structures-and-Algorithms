For Leetcode, you do not need all of STL. Around 20–25 functions + patterns cover most problems.

I’d group them like this:

⸻

1. Vector (Most used)

vector<int> v;

Insert / remove

v.push_back(x);
v.pop_back();
v.insert(v.begin()+i, x);
v.erase(v.begin()+i);
v.clear();

Access

v[i]
v.front()
v.back()
v.size()
v.empty()

Iteration

for(auto x : v)
for(int i=0;i<v.size();i++)

Algorithms

sort(v.begin(), v.end());
reverse(v.begin(), v.end());
max_element(v.begin(), v.end());
min_element(v.begin(), v.end());
accumulate(v.begin(), v.end(), 0);

Know:

lower_bound()
upper_bound()
binary_search()

⸻

2. String

string s;

Access

s[i]
s.length()
s.size()
s.empty()

Modify

s.push_back(c);
s.pop_back();
s.substr(start,len);
s.erase(pos,len);
reverse(s.begin(), s.end());

Conversion

stoi(s);
stol(s);
to_string(x);

Search

s.find("abc");
s.find(c);
s.compare(s2);

Useful:

isdigit(c)
isalpha(c)
tolower(c)
toupper(c)

⸻

3. Set / unordered_set

Use when:

unique elements
membership checking

Functions:

insert()
erase()
count()
find()
size()
empty()

Ordered set only:

lower_bound()
upper_bound()

⸻

4. Map / unordered_map

Use:

frequency
lookup
pair storage

Functions:

mp[key]
mp[key]++
mp[key]--
count()
find()
erase()
size()

Iteration:

for(auto [k,v] : mp)

Common pattern:

unordered_map<int,int> freq;
for(int x:nums)
    freq[x]++;

⸻

9. Pair

10. Sorting + Comparator

Default:

sort(v.begin(), v.end());

Descending:

sort(v.begin(), v.end(),
greater<int>());

Custom:

sort(v.begin(), v.end(),
[](auto a, auto b)
{
    return a.second<b.second;
});

Must know.

⸻

11. Binary Search helpers

lower_bound()
upper_bound()
binary_search()

Example:

vector<int> v={1,2,2,3};
lower_bound
→ first 2
upper_bound
→ after last 2

Distance:

idx=
lower_bound(
v.begin(),
v.end(),
x
)-v.begin();

13. Bit operations

Check bit:

Set:

Clear:

x^=(1<<i);

Count bits:

__builtin_popcount(x)
__builtin_popcountll(x)

Trailing zeros:

__builtin_ctz(x)
